# prompts/ — reusable session prompts

Project-specific adaptations of the [SafeVibeCoding](https://github.com/Frostbite1536/safe-vibe-coding) prompt library. Use the matching prompt at session start for consistent results.

| Prompt | Use |
|---|---|
| [engineering.md](engineering.md) | Default coding mode for roadmap phases |
| [bug-hunt.md](bug-hunt.md) | Periodic sweeps; run before each milestone review |
| [code-review.md](code-review.md) | Reviewing a diff/PR against REVIEW.md + INVARIANTS.md |
